# RechercheInformation

## Create venv

Create env and install packages

**Windows (Powershell)**

```python
python -m venv .venv
.venv/Scripts/activate.ps1
python -m pip install -r requirements.txt
```

**Unix (Bash)**

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```
